## Tentang CodeCinema

**CodeCinema** adalah aplikasi web pemesanan tiket bioskop yang dibangun menggunakan **Laravel Framework**.  
Aplikasi ini menyediakan fitur **manajemen film**, **jadwal tayang**, **lokasi bioskop**, dan **studio**, dengan tampilan modern serta sistem admin yang sederhana.

### Fitur Utama:
- Manajemen Film (Now Playing & Coming Soon)
- Pengaturan Jadwal Tayang
- Multi Lokasi Bioskop
- Multi Studio (Regular, Premiere, IMAX)
- Role Admin & User
